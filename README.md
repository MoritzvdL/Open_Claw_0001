# Kunden-Repository (Open_Claw_0001)

Dieses Repository dient dazu, Informationen zu **potenziellen Kunden** sauber und nachvollziehbar abzulegen.

## Struktur

- Pro Kunde ein Ordner unter `/Kunden/<Kundenname>/`
- Je Kunde:
  - `README.md` (Kurzprofil + Links)
  - `Business/` (Top-Initiativen & größere Änderungen)
  - `Technologie/` (Top-Initiativen & größere Änderungen)

## Konventionen

- Initiativen werden als einzelne Markdown-Dateien gepflegt: `YYYY-MM-DD__kurztitel.md`
- Jede Initiative enthält: Kontext, Status, Owner/Stakeholder (falls bekannt), nächste Schritte, Quellen/Links.
